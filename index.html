About

In brief: this tool compares two texts, even if they contain styling.

Full context: the tool was initially intended for a text-editor that has change-tracking function. I t assumes that the text changed in time, so we would typically compare:

original text

("before" version)

versus:
current text

("after" version)

Main challenge: it is actually comparing HTML strings, because text might be styled.

Malformed example:

<mark class="added"><b>bold </mark>text</b>
Minor challenges

Identify so-called corresponding changes, for example the middle word in those phrases:
Going surfing soon → Going skating soon
